== README


* A review app produced in primarily Ruby on Rails with 3 gems to help with search, star rating and bookings. 
* Following gems where used
  - Ransack
  - Bookable
  - Ratyrate

* The user should be able to search for a PT by skill-set and location.  
* The user can also read reviews and add them
* PT can also add themselves to the app and accept bookings
